void client::Client::createPlayerEntity ( NewPlayerPacket packet)

Creates and configures the player entity in the ECS.

Constructs a player entity and attaches its initial components: position, velocity, movement speed, render asset, sprite source rectangle, scale, player tag, and sprite animation metadata.

The created entity is positioned at (100, 100) with zero initial velocity and uses renderManager::PLAYER_PATH for rendering. Sprite and scale values are taken from PlayerSpriteConfig. The sprite animation component is initialized with column/row counts, selected/neutral frames, frame timing, and non-playing, non-looping defaults.
