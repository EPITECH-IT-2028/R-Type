Update per-\/entity movement velocity and vertical sprite animation state based on keyboard input. 

For each entity in the system, reads up/down/left/right key state to compute a normalized movement direction, assigns the entity\textquotesingle{}s velocity by multiplying the direction by its \doxylink{structecs_1_1_speed_component}{Speed\+Component}, and updates the entity\textquotesingle{}s \doxylink{structecs_1_1_sprite_animation_component}{Sprite\+Animation\+Component} vertical state\+:
\begin{DoxyItemize}
\item When UP (and not DOWN) is pressed, ensures animation plays forward (positive frame\+Time) from the neutral frame.
\item When DOWN (and not UP) is pressed, ensures animation plays backward (negative frame\+Time) from the neutral frame.
\item When neither or both vertical keys are pressed, stops the animation and resets to the neutral frame with non-\/negative frame\+Time.
\end{DoxyItemize}

Also handles shooting input\+: when the space bar is pressed, calls the client\textquotesingle{}s send\+Shoot method with the local player\textquotesingle{}s position.
