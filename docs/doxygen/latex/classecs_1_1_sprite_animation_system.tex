Advance sprite animations for all tracked entities and update their source rectangles. 

Advances each entity\textquotesingle{}s \doxylink{structecs_1_1_sprite_animation_component}{Sprite\+Animation\+Component} based on the elapsed time and the component\textquotesingle{}s configuration\+: when the animation is playing and {\ttfamily frame\+Time} is non-\/zero, accumulated time is used to move the current frame forward (positive {\ttfamily frame\+Time}) or backward (negative {\ttfamily frame\+Time}). When the animation passes its start/end bounds it will either loop back to the configured start/end frame if {\ttfamily loop} is true, or clamp to the final frame if {\ttfamily loop} is false. After advancing frames, the sprite\textquotesingle{}s {\ttfamily source\+Rect} is refreshed from the current frame.
