\doxysubsection{Detailed Description}
Describes an enemy death event sent from server to client. 

Starts with a \doxylink{struct_packet_header}{Packet\+Header} identifying the packet type and payload size.

Fields\+:
\begin{DoxyItemize}
\item header\+: \doxylink{struct_packet_header}{Packet\+Header} present in all packets.
\item enemy\+\_\+id\+: Server-\/assigned identifier for the enemy that died.
\item death\+\_\+x\+: X coordinate of the death location in world space.
\item death\+\_\+y\+: Y coordinate of the death location in world space.
\item player\+\_\+id\+: Identifier of the player credited for the kill.
\item score\+: Score awarded for the kill. 
\end{DoxyItemize}
