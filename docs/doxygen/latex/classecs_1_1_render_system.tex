Renders all entities managed by this system, loading and caching textures and initializing sprite animations when needed. 

Iterates over tracked entities and, for each one with a non-\/empty texture path, ensures the texture is loaded and cached, initializes \doxylink{structecs_1_1_sprite_animation_component}{Sprite\+Animation\+Component} frame dimensions if the component is present and not initialized, constructs source and destination rectangles (respecting \doxylink{structecs_1_1_sprite_component}{Sprite\+Component} source rects, \doxylink{structecs_1_1_render_component}{Render\+Component} size/offsets, \doxylink{structecs_1_1_background_tag_component}{Background\+Tag\+Component} fullscreen-\/aspect behavior, and \doxylink{structecs_1_1_scale_component}{Scale\+Component}), and issues the draw call for the computed rectangles.

Observable side effects\+:
\begin{DoxyItemize}
\item Loads textures from disk and stores them in the system\textquotesingle{}s texture cache.
\item Logs a warning and skips rendering if a texture fails to load or has zero height for background entities. 
\end{DoxyItemize}
