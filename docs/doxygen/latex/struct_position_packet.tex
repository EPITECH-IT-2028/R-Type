\doxysubsection{Detailed Description}
Sends the client\textquotesingle{}s current position with an ordering sequence number. 

Packet used by the client to report its position to the server. From server to client.

Fields\+:
\begin{DoxyItemize}
\item header\+: Common packet header identifying the packet type and payload size.
\item sequence\+\_\+number\+: Client-\/side sequence number for ordering position updates.
