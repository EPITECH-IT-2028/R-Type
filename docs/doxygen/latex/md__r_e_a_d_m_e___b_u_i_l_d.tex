\begin{DoxyItemize}
\item {\ttfamily BUILD\+\_\+\+CLIENT}\+: Build the client (OFF by default)
\item {\ttfamily BUILD\+\_\+\+SERVER}\+: Build the server (OFF by default)

\end{DoxyItemize}

At least one of {\ttfamily BUILD\+\_\+\+CLIENT} or {\ttfamily BUILD\+\_\+\+SERVER} must be ON.
